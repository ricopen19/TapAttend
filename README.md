# TapAttend

iPad でタップして出席管理できる Web アプリ。
紙の教務手帳の「集計がめんどくさい」を解決します。

## できること

- 表形式の名簿にタップで出席状況を入力（出席・欠席・遅刻・早退・公欠・忌引・出停）
- 「＋」ボタンで今日の日付の列を追加（日付の手動修正も可能）
- 長押しで備考入力（公欠の理由など）
- 出席数・欠席数・出席率を自動計算
- CSV エクスポート（Excel に貼れる形式）
- JSON バックアップ / リストア
- オフラインでも動作（PWA）

## データについて

入力したデータは**あなたのブラウザの中だけ**に保存されます。サーバーには一切送信されません。
詳しくは [データの取り扱いについて](DATA_POLICY.md) をご覧ください。

**Tips**: 生徒名はイニシャルや出席番号だけでも運用できます（例：「A・H」「1番」）。
個人情報が気になる場合はこの方法がおすすめです。

## 使い方

### 1. クラスを作る

ホーム画面でクラス名を入力して「追加」。
例：「1-5 数学I」「2-3 英語II」

### 2. 生徒を登録する

クラスの「生徒」ボタンから、出席番号と氏名を入力して追加。
CSV ファイル（番号,氏名 の形式）での一括取り込みにも対応しています。

### 3. 出席を入力する

クラス名をタップすると出席簿が開きます。

| 操作 | 動作 |
|---|---|
| 「＋ 授業日追加」 | 今日の日付で列を追加 |
| セルをタップ | 出席 → 欠席 → 遅刻 → 早退 → 公欠 → 忌引 → 出停 → 出席... の順に切り替え |
| セルを長押し | 備考を入力（備考があるセルは青枠で表示） |
| 日付をタップ | 日付を修正 |
| 日付下の「×」 | その授業日の列を削除 |

右端の列に出席数・欠席数・出席率がリアルタイムで表示されます。

### 4. データを出力する

- **CSV出力**: Excel で開ける出席簿形式のファイルをダウンロード
- **バックアップ**: 全データを JSON ファイルで保存（定期的に取ることを推奨）
- **リストア**: バックアップファイルからデータを復元

## 動作環境

| デバイス | ブラウザ |
|---|---|
| iPad（推奨） | Safari / Chrome |
| iPhone | Safari / Chrome |
| PC（Windows / Mac） | Chrome |

ホーム画面に追加するとアプリのように使えます（PWA）。
